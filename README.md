# BAFL-Backend

This is the backend for the BAFL website for the admins and coaches.

## Python Version

This project uses **Python 3.12** (the latest stable version). All developers must use Python 3.12 for consistency.

**We strongly recommend using Conda for environment management.** See [CONDA_SETUP.md](CONDA_SETUP.md) for detailed setup instructions.

## Quick Setup

```bash
# Using Conda (recommended)
conda env create -f environment.yml
conda activate bafl-backend

# Verify Python version
python --version  # Should show 3.12.x
```
